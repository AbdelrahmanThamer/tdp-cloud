# 源码目录结构解析

## 下表中 `*talbe*` 表示和数据表名一一对应的文件或文件夹

```shell
TDP Cloud
│
├─api ------------------------------------ HTTP 接口
│  ├─cloudflare -------------------------- Cloudflare 路由及控制器
│  ├─tencent ----------------------------- 腾讯云路由及控制器
│  ├─terminal ---------------------------- SSH 代理路由及控制器
│  ├─workhub ----------------------------- 节点服务路由及控制器
│  └─*talbe* ----------------------------- *talbe*表路由及控制器
│
├─cmd ------------------------------------ CLI 命令
│  ├─args -------------------------------- 共享参数表
│  └─subset ------------------------------ 子命令集合
│
├─docs ----------------------------------- 说明文档
│
├─docker --------------------------------- Docker 构建工具
│
├─front ---------------------------------- 前端界面
│
├─helper --------------------------------- 辅助包
│  ├─alibaba ----------------------------- 阿里云接口集成
│  ├─certmagic --------------------------- CertMagic 集成
│  ├─cloudflare -------------------------- Cloudflare 接口集成
│  ├─command ----------------------------- 系统命令工具
│  ├─psutil ------------------------------ 系统监控工具
│  ├─request ----------------------------- HTTP 客户端
│  ├─socket ------------------------------ Socket 服务集成
│  ├─strutil ----------------------------- 字符串处理函数
│  ├─tencent ----------------------------- 腾讯云接口集成
│  └─webssh ------------------------------ WebSSH 集成
│
├─module --------------------------------- 内部模块
│  ├─certbot ----------------------------- 证书管理服务
│  ├─crontab ----------------------------- 计划任务服务
│  ├─dborm ------------------------------- 数据库操作
│  │  ├─passport ------------------------- 用户中心模型
│  │  └─*talbe* -------------------------- *talbe*表模型
│  ├─httpd ------------------------------- HTTP 服务器
│  ├─midware ----------------------------- HTTP 中间件
│  ├─migrator ---------------------------- 数据库自动迁移
│  ├─worker ------------------------------ 子节点服务
│  └─workhub ----------------------------- 中心节点服务
│
├─service -------------------------------- 服务管理
│  ├─server ------------------------------ 中心节点服务管理
│  └─worker ------------------------------ 子节点服务管理
│
└─tests ---------------------------------- 接口测试
```
