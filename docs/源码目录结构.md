# 源码目录结构解析

## 一级目录及部分二级目录说明

```shell
TDP Cloud
│
├─api ----------------------------- HTTP 接口
│
├─cmd ----------------------------- CLI 命令
│  ├─args ------------------------- 共享参数表
│  └─subset ----------------------- 子命令集合
│
├─docs ---------------------------- 说明文档
│
├─front --------------------------- Web 界面
│
├─helper -------------------------- 辅助包
│  ├─cloudflare ------------------- Cloudflare 接口集成
│  ├─psutil ----------------------- 操作系统监控工具类
│  ├─qcloud ----------------------- 腾讯云接口集成
│  ├─request ---------------------- HTTP 客户端
│  ├─socket ----------------------- Socket 服务集成
│  ├─strutil ---------------------- 字符串处理函数
│  └─webssh ----------------------- Web SSH 集成
│
├─module -------------------------- 内部模块
│  ├─cron ------------------------- 计划任务
│  ├─dborm ------------------------ 数据库及模型
│  ├─httpd ------------------------ HTTP 服务器
│  ├─midware ---------------------- HTTP 中间件
│  ├─migrator --------------------- 数据库迁移
│  ├─worker ----------------------- 子节点服务
│  └─workhub ---------------------- 中心节点服务
│
├─service ------------------------- 服务管理
│  ├─server ----------------------- 中心节点服务管理
│  └─worker ----------------------- 子节点服务管理
│
└─tests --------------------------- 接口测试
```
