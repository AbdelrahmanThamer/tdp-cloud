# 绑定腾讯云

1、进入 **腾讯云** [CAM - 策略](https://console.cloud.tencent.com/cam/policy) 页面，创建一个自定义策略 `TDPCloudAccess`，权限JSON如下：

```json
{
    "Version": "1",
    "Statement": [
        {
            "Action": [
                "alidns:*",
                "ecs:*",
                "ram:GetUser",
                "swas:*",
                "vpc:DescribeVpcs",
                "vpc:DescribeVSwitches"
            ],
            "Resource": "*",
            "Effect": "Allow"
        }
    ]
}
```
